# 发布说明 - feat/plan1-commission-engine

## 🎯 功能概述

完整的牛奶销售提成计算系统，从数据导入到工资表生成的全流程自动化。

## ✨ 核心功能

### 提成计算引擎
- **3维比例表**：门店类别 × 达成档位 × 商品档位
- **达成率计算**：按「人×店」分别计算，不跨店合并
- **商品档位**：常温/低温 + 高毛/低毛/特价（按毛利率自动判定）
- **赠送处理**：精确剔除（订单号+条码匹配）
- **退货冲抵**：源单号→小票单号+条码精确匹配（97.7%匹配率）

### Web 管理后台
- **商品档案**：286条商品，支持批量标记「不计提成」、批量删除
- **门店信息**：35个门店，按组批量改类别
- **月度工作台**：4步流程
  1. 导入数据（销售流水 + 赠送明细）
  2. 配置目标（门店月度目标）
  3. 当班确认（自动推断 + 手动调整冲突）
  4. 计算&结果（一键计算 + Top5排行 + 导出Excel）

### 技术栈
- **后端**：Python 3.11 + FastAPI + SQLAlchemy + SQLite
- **前端**：React 18 + Vite + TypeScript + shadcn/ui + Tailwind CSS
- **设计**：Notion风格，金融语义配色（绿色表提成/收益）
