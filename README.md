# 牛奶提成计算引擎（Plan 1）

独立 Python 引擎，按业绩工资制度算牛奶销售提成，导出工资表+明细。

## 安装
    pip install -e ".[dev]"

## 跑当月
    salary-engine run \
      --sales 6月历史销售流水.xlsx \
      --gifts 6月历史零售让利明细.xlsx \
      --products-info 商品信息表（常温低温）.xlsx \
      --cost 销售成本.xlsx \
      --stores-file 分组.xlsx --stores-sheet "2026.6全部（正确）" \
      --month 2026-06 --days 30 --out 工资表.xlsx

## 测试
    pytest -q

## 验收对账（关键）
跑完 2026年6月 后，把 `工资表` sheet 与手工 `分组.xlsx` 的结果逐人对账。
若金额不一致，优先排查：成本条码匹配、门店名前缀剥离、当班推断、退货匹配。
引擎会在 stdout 打印 warnings（缺成本/未知门店/退货异常）供定位。

## 说明
- 提成 = 销售额 × 提成比例；比例 = 查3维表(门店类别A/B/C/D × **个人月度达成率档** × 商品档位)。
- 个人月度达成率 = 该人当月全部当班天的乳品业绩之和 ÷ 这些天对应门店目标之和(各店月目标÷当月天数，按当班天累加，跨店也合并)。
- 商品档位 = 常温/低温 × 高毛/低毛/特价（按本笔毛利率判定，阈值：常温17%/10%、低温15%/10%；恰好10%归特价）。
- 赠送：让利明细整份即赠送清单，按(单号,条码)从销售流水剔除。
- 退货：精确匹配(源单号=小票单号+条码)冲抵原行；不匹配的算到退货当日、提成负数、标黄。
- 当班人取自销售流水『收银员名称』（『营业员名称』为空）。
